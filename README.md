# ETF Arbitrage Simulator

A Python-based limit order book simulator and ETF arbitrage playground inspired by trading challenges.

## Goals
- Simulate a live exchange with limit and market orders
- Implement ETF–ETF and ETF–basket arbitrage strategies
- Track PnL, inventory, and risk in real time

## Structure
- `src/` – exchange, market generator, strategies
- `tests/` – unit tests
- `notebooks/` – experiments and analysis
